YOUTUBE AUTOMATION PROJECT - VIDEO ASSEMBLER
============================================
File: core/assembler.py

Purpose
-------
VideoAssembler turns voiced scenes and downloaded visuals into the final vertical video.

Main Class
----------
VideoAssembler

Input
-----
The assembler looks for one MongoDB task with status:

ready_to_assemble

Required task fields:
- script_data
- folder_path
- title
- niche

Current Behavior
----------------
- Loads Whisper to get word timestamps from each scene audio file.
- Loads CLIP when available to choose strong video start points.
- Falls back to OpenCV-based analysis when CLIP is unavailable.
- Applies niche-specific color grading and film grain.
- Builds a 24 FPS vertical video from scene visuals and audio.
- Draws synchronized captions over the base video.
- Writes FINAL_VIDEO.mp4.
- Cleans intermediate files while preserving FINAL_VIDEO.mp4 and FINAL_VIDEO_METADATA.txt.

Key Dependencies
----------------
- moviepy
- Pillow
- numpy
- openai-whisper
- torch
- rapidfuzz
- optional CLIP installed from GitHub

Key Methods
-----------
- _load_clip() / _unload_clip()
  Manage optional CLIP model memory.
- _load_whisper() / _unload_whisper()
  Manage Whisper model memory.
- _find_best_start(video_path, required_duration, keyword="")
  Selects a useful starting point for source videos.
- _apply_color_grade(...)
  Applies niche-specific visual style.
- _make_clip(...)
  Converts a media path into a timed MoviePy clip.
- _write_base_video(...)
  Creates the base video and audio mix.
- _draw_text_on_video(...)
  Adds synchronized text/captions.
- _cleanup_intermediate_files(...)
  Removes raw assets after final render.
- assemble()
  Full assembly entry point.

Output
------
Writes:

FINAL_VIDEO.mp4

Then updates MongoDB with:
- status: ready_to_upload
- final_video_path
