YOUTUBE AUTOMATION PROJECT - NEWS SCRAPER
=========================================
File: core/scraper.py

Purpose
-------
NewsScraper discovers video topics. It rotates through niches, fetches RSS stories, asks the AI engine to choose three viral candidates, deep-reads the chosen article, and saves a pending task in MongoDB.

Main Class
----------
NewsScraper

Key Methods
-----------
- get_time_slot()
  Maps the current hour to one of the six project slots.
- fetch_rss(url)
  Fetches up to 10 RSS entries from one feed.
- pick_top_3_viral_topics(candidates, niche)
  Uses AIEngine to rank candidates for YouTube Shorts potential. If AI fails, it randomly samples candidates as a fallback.
- extract_full_article(url)
  First tries Jina Reader, then falls back to BeautifulSoup paragraph extraction.
- refine_user_idea(topic, content, feedback="")
  Rewrites a manual topic into a factual source article for the script generator.
- fetch_and_present_topics(slot)
  Selects an unused niche for the day, fetches feeds in parallel, filters recent duplicates, and returns three topic choices.
- save_approved_topic(chosen_topic, niche, niche_data, slot)
  Saves the selected story as a pending MongoDB task.
- scrape_targeted_niche(forced_slot=None)
  Automatic-mode helper that chooses the first AI-ranked topic.

Niches
------
The scraper currently supports these niche groups:
- war_news
- space
- tech_ai
- finance_economy
- bizarre_facts
- psychology
- health_wellness
- animals_nature

Each niche defines RSS feeds, default visual style, hashtags, and voice model.

Duplicate Protection
--------------------
Before saving a task, DBManager checks for:
- The same source URL used in the last 7 days.
- A title with more than 85 percent similarity to a recent task.

Output
------
Successful scraping creates a MongoDB task with status:

pending

Important Fields Saved
----------------------
- title
- content
- source
- source_url
- niche
- slot
- folder_path
- pexels_style
