YOUTUBE AUTOMATION PROJECT - UPLOAD PREP
========================================
File: core/upload_prep.py

Purpose
-------
UploadManager writes a clean metadata file beside the final video and advances the task so YouTubeUploader can pick it up.

Main Class
----------
UploadManager

Input
-----
The manager looks for one MongoDB task with status:

ready_to_upload

Required task fields:
- folder_path
- title
- ai_description
- ai_tags

Current Behavior
----------------
1. Verifies the task folder exists.
2. Reads title, description, and tags from the task.
3. Normalizes tags whether they are stored as a list or a string.
4. Writes FINAL_VIDEO_METADATA.txt inside the generated-video folder.
5. Updates MongoDB status to completed_packaged.

Output File
-----------
FINAL_VIDEO_METADATA.txt contains:
- TITLE
- DESCRIPTION
- TAGS

Status Output
-------------
On success:

completed_packaged

Notes
-----
The file is intentionally named FINAL_VIDEO_METADATA.txt so assembler cleanup preserves it.
